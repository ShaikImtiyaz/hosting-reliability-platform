# Cloud Native Hosting Reliability Platform

## Overview

This repository contains a production-style Hosting Reliability Platform built to demonstrate modern Site Reliability Engineering (SRE), DevOps, and Cloud Infrastructure practices.

The project showcases end-to-end infrastructure automation, container orchestration, centralized monitoring, self-healing automation, and CI/CD deployment workflows.

## Technology Stack

- Python
- Flask
- Docker
- Kubernetes (AKS)
- Microsoft Azure
- Terraform
- Ansible
- Elasticsearch
- Logstash
- Kibana
- GitHub Actions
- Bash
- Linux

## Features

- Application health monitoring
- Infrastructure automation
- Kubernetes deployment
- Centralized logging
- Metrics collection
- Self-healing automation
- CI/CD pipeline
- Production monitoring
- Infrastructure as Code (Terraform)
- Configuration Management (Ansible)
